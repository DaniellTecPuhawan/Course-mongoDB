Operadores de comparación
$eq: Igual a (Equal to)
$ne: No igual a (Not equal to)
$gt: Mayor que (Greater than)
$gte: Mayor o igual que (Greater than or equal to)
$lt: Menor que (Less than)
$lte: Menor o igual que (Less than or equal to)
$in: Coincide con cualquiera de los valores en la lista (Matches any values in a list)
$nin: No coincide con ninguno de los valores en la lista (Matches none of the values in a list)
Operadores lógicos
$and: Todos los criterios deben coincidir (All criteria must match)
$or: Al menos uno de los criterios debe coincidir (At least one criterion must match)
$not: Niega la condición (Negates the condition)
$nor: Ninguno de los criterios debe coincidir (None of the criteria must match)
Operadores de elementos
$exists: Verifica si el campo existe (Checks if a field exists)
$type: Selecciona documentos donde el campo es de un tipo específico (Matches documents where the field is of a specific type)
Operadores de evaluación
$mod: Divide un valor por un divisor y compara el resto (Performs modulo operation)
$regex: Coincide con una expresión regular (Matches regular expression)
$text: Realiza una búsqueda de texto (Performs text search)
$where: Evalúa una expresión JavaScript en los documentos (Matches based on JavaScript expression)
Operadores de arreglo (array)
$all: Coincide si un arreglo contiene todos los elementos especificados (Matches arrays containing all elements)
$elemMatch: Coincide si al menos un elemento en un arreglo satisface la condición (Matches if at least one element in an array matches the condition)
$size: Coincide si el arreglo tiene un número específico de elementos (Matches arrays with a specific number of elements)
Operadores de actualización
$set: Establece el valor de un campo (Sets the value of a field)
$unset: Elimina un campo de un documento (Removes a field from a document)
$inc: Incrementa el valor de un campo (Increments the value of a field)
$push: Añade un valor a un arreglo (Adds a value to an array)
$pop: Elimina el primer o último elemento de un arreglo (Removes the first or last element of an array)
$pull: Elimina todos los elementos coincidentes de un arreglo (Removes all matching elements from an array)
$addToSet: Añade un valor a un arreglo solo si no existe (Adds a value to an array if it doesn't already exist)
Operadores de agregación
$sum: Suma de valores (Sums values)
$avg: Calcula el promedio de los valores (Calculates the average of values)
$min: Encuentra el valor mínimo (Finds the minimum value)
$max: Encuentra el valor máximo (Finds the maximum value)
$first: Selecciona el primer valor en un grupo (Selects the first value in a group)
$last: Selecciona el último valor en un grupo (Selects the last value in a group)
$count: Cuenta el número de documentos que cumplen el criterio (Counts the number of documents that match)
$group: Agrupa documentos por un campo específico (Groups documents by a specific field)
$project: Cambia la forma de los documentos que entran en la canalización (Reshapes the documents that enter the pipeline)
Operadores de bit a bit
$bitsAllClear: Coincide si ninguno de los bits especificados está establecido (Matches if none of the specified bits are set)
$bitsAnyClear: Coincide si al menos uno de los bits especificados no está establecido (Matches if any of the specified bits are clear)
$bitsAllSet: Coincide si todos los bits especificados están establecidos (Matches if all the specified bits are set)
$bitsAnySet: Coincide si al menos uno de los bits especificados está establecido (Matches if any of the specified bits are set)
Otros operadores importantes
$lookup: Realiza una operación de "join" entre dos colecciones (Performs a join between two collections)
$merge: Combina los resultados de la agregación en una colección (Merges the aggregation results into a collection)
